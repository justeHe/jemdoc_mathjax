Inline-Level Elements

Inline elements format text within block-level elements.

Text Formatting

  • Bold: *text* -> text

  • Italic: /text/ -> text

  • Underline: _text_ -> text

  • Monospace: +text+ -> text

  • Quotes: "text" -> “text”

Links

  • External Link: [http:jemdoc.jaboc.net Jemdoc website] -> Jemdoc website

  • Internal Link: [/index.html Home] -> Home

  • Email Link: [name@example.com] -> name@example.com

Images

  • Jemdoc Syntax: [img{width}{height}{alt_text} image_url]

  • Example: [img{100px}{}{Logo} docsimage.png]
