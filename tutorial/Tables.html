Tables

Jemdoc supports creating tables using a specific block syntax.

  • Jemdoc Syntax: 

~~~
{}{table}{my-table}
Header 1 | Header 2 | Header 3 ||
Row 1, Col 1 | Row 1, Col 2 | Row 1, Col 3 ||
Row 2, Col 1 | Row 2, Col 2 | Row 2, Col 3
~~~

  • Display:

Header 1 Header 2 Header 3
Row 1, Col 1 Row 1, Col 2 Row 1, Col 3
Row 2, Col 1 Row 2, Col 2 Row 2, Col 3
